# Disappear

Free your desktop from cumbersome apps.

## Install

```sh
git clone https://github.com/sonnyp/Disappear.git
cd Disappear
sudo ./install.sh
```

## Update

```sh
cd Disappear
sudo ./uninstall.sh
git pull
sudo ./install.sh
```

## Uninstall

```sh
cd disappear
sudo ./uninstall.sh
```
